🌍 Earthquake Impact Predictor

🔗 Live App: https://impactsense-zvvchrnc6atp3xihqcuqcp.streamlit.app/

📌 Overview

Earthquake Impact Predictor is a web-based application that estimates earthquake risk based on key seismic parameters such as magnitude, depth, and intensity measures.

Built using Streamlit, the app provides an intuitive and visually appealing interface for quick predictions and experimentation.

Streamlit enables rapid development and deployment of data-driven web apps directly using Python, making it ideal for ML and analytics projects.

🚀 Features
📊 Predict earthquake impact based on:
Magnitude
Depth (km)
CDI (Community Determined Intensity)
MMI (Modified Mercalli Intensity)
Significance (SIG)
🎨 Modern UI with gradient design
⚡ Fast and interactive input handling
🌐 Deployed and accessible online
🛠️ Tech Stack
Frontend & Backend: Streamlit
Language: Python
Deployment: Streamlit Community Cloud
🧠 How It Works
Enter seismic parameters into the input fields
Click “Predict Risk”
The model processes the inputs and returns a prediction
